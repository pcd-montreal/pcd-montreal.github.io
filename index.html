// ABOUT



Processing Community Day Montréal 2019 is an inclusive event to celebrate art, code, and diversity. It features presentations from various people who write code opening to its creative potential, workshops related to creative coding, open source software and live coding. This event is part of a large network of similar events that will happen around the world (in Bangalore, Los Angeles, Madrid, São Paulo, Stockholm, Tokyo, etc.), brought together by the Processing Foundation, a non-profit organization promoting software literacy within the visual arts, and visual literacy within technology-related fields — and dedicated to making these fields accessible to diverse communities.



// LOCATION



February 9, 2019 at Concordia University (EV.6720), thanks to the support of the Department of Design and Computation Arts.
